06 — Charts

กราฟ

SVG smooth bezier area — เส้นโค้ง cubic bezier (cardinal spline) · gradient fill · dashed gridlines · คำนวณ server-side ใน Astro frontmatter ไม่ต้องใช้ charting lib · column/bar และ pie ก็ใช้ SVG เช่นกัน — ไม่มี charting library

Axis label pattern — Y-axis และ X-axis เป็น HTML ทั้งคู่ ไม่ใช้ SVG <text> เพราะ preserveAspectRatio="none" จะทำให้ตัวอักษรยึดตามกว้างหน้าจอ · Y-axis: div[relative h-40 width:1.75rem] + span[absolute right-0] top % คำนวณจาก (MT + (1−f)×ph) / VH × 100 · X-axis: flex justify-between ใต้ SVG, padding-right: MR/VW% (ML=0 แล้ว) · SVG ตั้ง ML=0 เพราะ Y label อยู่ใน HTML column แทน
